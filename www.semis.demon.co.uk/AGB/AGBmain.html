GameBoy ADVANCE
Development

There is very little technical information available for the AGB system. So I put a few random ideas together to see what would be needed to construct a simple ROM and RAM memory interface. This is what I came up with AGBmem.bmp. No idea if it would actually work but I think it contains all the basic building blocks of a 'random' and 'sequential' access interface. It assumes the use of a separate FLASH programmer to put the development program into the ROM. Please don't ask me questions about how it works or other AGB interfacing things, if you don't understand it then I should try something else.
