2000-12-26 - BootViewer v1.1 - Posted by: CyBuilder
This application allows you replace the boot logo with a photo 'logo.pic' that you can create yourself. Hold down <Enter> to view the boot logo as long as you want. Here are installation instructions. NOTE: IF YOU COPY THIS FILE TO YOUR OWN WEB PAGE YOU MUST PROVIDE A LINK TO http://www.devrs.com/cybiko SOMEWHERE SINCE THE LATEST VERSIONS ARE OBTAINED HERE.

2001-01-10 - Cybi-Go - Posted by: Jeff Frohwein
This windows program (along with source code) allows you to download web content straight from the web to your cybiko. It seems to extract words from web pages for viewing on the cybiko.

2000-11-13 - CyMail - Posted by: Jeff Frohwein
CyMail is a windows program to download email from your POP3 server onto your cybiko.

2001-01-02 - CyUCon v0.1231 - Cybiko Unix Console - Posted by: Jeff Frohwein
This application for linux (including source code) allows you to send games/apps/text to your cybiko. There is also support for executing console commands.

2001-01-22 - VTTerm v1.2 - Posted by: Jeff Frohwein
Here's a VT100 Terminal emulator for Cybiko. Only cursor positioning & clear screen escape sequences are currently supported. You can also change the font size between 3x5,4x6, & 5x7 as well as the baud rate, parity, etc. Source code is included.
