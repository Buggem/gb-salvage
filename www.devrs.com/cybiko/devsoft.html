Apps

2001-01-14 - AppHack - Posted by: Jeff Frohwein
AppHack allows browsing of the Cybiko File system, and enables you to browse and export the resources within .app archives. Written by Ernest S. Pazera

2001-04-03 - Cybiko Debugger v1.0402 - Posted by: Jeff Frohwein
Here's an app that allows you to view memory, patch memory, and search for a string. It must be used in SDK Console. No desktop icon will appear for it. After unzipping & sending it to the cybiko using SDK Console, type 'debug' in the console bottom line to start it up.

2001-01-14 - FontMake v1.0 - Posted by: Jeff Frohwein
Cybiko fixed width font making utility for WIN32. Full VC++ source included. It's currently only useful for Cybiko developers making their own games/apps. Written by Ernest S. Pazera.

Graphics

2001-01-14 - Screen Capture - Posted by: Jeff Frohwein
Screen Capture function that can be included in your Cybiko game or app. Copy the resulting file to your PC and use CyberLoad to convert it to a .bmp file. Great for web page screen previews. Written by Ernest S. Pazera.

His implementation of the standard I/O functions for the Cybiko: stdio.c & stdio.h.
