2000-05-16 - 2/3 Wire LCD Interface - Posted by: Jeff Frohwein
Here are some alternative designs to Myke Predko's 2 Wire LCD interface design that I've put together. The first design uses 2 wires and a 74HC595 and an RC circuit instead of a 74LS174. With this setup you gain 2 extra digital output lines. This design requires a delay period after sending bits for the RC to rise to Vcc and latch the data bits. The 3 wire designs require no delay period but require an extra line. BUT.. this extra lost line is not a total loss in that you gain 3 extra digital output lines by using this method. Note that all of these designs allow unlimited outputs through chaining if you can afford the extra time required to shift load each register. Also, you can update the LCD only, output ports only, or both at the same time by sending a new bit stream.
