2002-02-02 - GP32 CPU Datasheet & tools - Posted by: Jeff Frohwein
SAMSUNG's S3C2400X01 16/32-bit ARM920T RISC microprocessor. To reduce total system cost, S3C2400X01 also provides the following : separate 16KB Instruction and 16KB Data Cache, MMU to handle virtual memory management, LCD controller(STN & TFT), 2-channel UART with handshake, 4-channel DMA, System Manager(chip select logic, EDO/SDRAM controller), 4-channel Timers with PWM, I/O Ports, RTC, 8-channel 10-bit ADC, IIC-BUS interface, IIS-BUS interface, USB Host, USB Device, Multi-Media Card Interface, SPI and PLL for clock generation.

2002-04-05 - GP32 Specs - Posted by: Jeff Frohwein
COST(~US$): Main System=180, Games=35

SCREEN: 320x240 3.8" TFT Reflective LCD, 64k colo(u)rs

SOUND: 16bit PCM, 4 Channel mixing, MP3 support, Midi/Wave stereo, 2 speakers, earphone jack.

PROCESSOR: 32bit ARM920T processor (CPU speed selectable: ~67MHz is standard, up to 130MHz when running from internal cache only seems possible), 16kbytes Instruction cache, 16kbytes Data cache, RTOS (Can be disabled for max performance), 8MByte SDRAM, 512KBytes Flash ROM

SMC INTERFACE: SMC memory card slot (For storing games, MP3s, files, etc.)

EXT INTERFACE: IIC-bus, etc.

USB INTERFACE: For moving files PC <-> GP32 and internet multiplayer games.

SIZE: 157mm wide , 71mm high, 28mm thick (6.2 x 2.9 x 1.1 inches)

POWER: 2AA batteries should last 12 hours.

CHIPS: Samsung S3c2400x01-eer0 CPU, Samsung LPC3600 LCD timing controller, Hynix HY57V641620HG 8mbyte (16bit wide) SDRAM, Atmel AT49BV040-90VC 512k byte (8bit wide) bios flash, Samsung S524LB0X91 32kbit IIC-bus eeprom, Phillips UDA1330A stereo filter IIS-bus DAC, 74HC14, LM324, Samsung LTS350Q1 LCD Display
