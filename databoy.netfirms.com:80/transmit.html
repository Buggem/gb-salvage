Transmit Page
From this page the user can send a canned test message to a receiving device such as a PC. Prior to entering this page the user must set the serial communication parameters using the Setup Page. Place the TXD and/or RXD DIP switch in the ON position to send the message on the corresponding RS-232 line. Press the Up-arrow key to send the test message once. Press the Down-arrow key to send the test message repeatedly.

Note: Prior to turning on RXD or TXD switches, the user must make sure the line is not being actively driven by another device.
