Data Scope in Freeze Mode
ASCII display mode Hexadecimal display mode
Freeze mode is activated by pressing the 'A' key in the live Data Scope page. The two screenshots above show ASCII and hexadecimal views of the same data. Captured data can be reviewed by moving the cursor with the arrow keys. In this example the cursor is located below the center of the screen. As the cursor moves, the following items of information, displayed in red, are updated in the bottom two rows of the screen:
  • The cursor location as a character count (1-N) from the start of the buffer. In this example the cursor is on character number 289.
  • The cursor location as a percentage of the total number of characters captured (44% in this example).
  • If enabled on the Setup Page, the time-stamp of the character at the cursor is displayed, in milliseconds, relative to the first character in the buffer. In this example the time-stamp is 475ms. Note that time-stamps are available only on DataBoy Plus.
  • The hexadecimal value of the character at the cursor (73 in this example).
  • The ASCII representation of the character at the cursor ('s' in this example).

Note: These screenshots were made using the NO$GMB Game Boy emulator.
