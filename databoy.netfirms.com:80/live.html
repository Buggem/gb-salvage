ASCII display mode Hexadecimal display mode
The two screenshots above show ASCII and hexadecimal views of the same captured data in the Live Data Scope page. The data display area of the screen (white background) is organized as 16 alternating rows of black and blue characters, representing RXD and TXD data, respectively. Each row displays up to 20 characters. The last displayed character is followed by a rectangular black cursor. When the cursor reaches the lower-right corner of the screen, it will wrap to the upper-left corner. Two prompts are displayed at the bottom of the screen . Pressing the 'A' key will stop data capture and start the Freeze mode. Pressing the 'B' key will toggle between ASCII and hexadecimal display modes. The extended IBM character set is used in ASCII mode.

Note: these screenshots was made using the NO$GMB Game Boy emulator.
