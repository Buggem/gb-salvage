Click on "Load & Run" to start the demo. Once the demo is running, use the keyboard to simulate the Game Boy buttons, according to the key assignments shown above. If there is no response to keystrokes, click on the Game Boy image (above) to give it focus.

This demo uses a modified version of GameBoyEmu, a Game Boy emulator written in Java. The author of this nifty applet is David Winchurch, who has generously made it available for others to use. For information about GameBoyEmu visit:

http://www.angelfire.com/ill/gbcroms/
