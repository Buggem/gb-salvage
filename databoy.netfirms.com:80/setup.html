Setup Page
The following parameters are set on this page.

Baud rate: This is the baud rate used in the Data Scope page and the Transmit page. The options are: 110, 150, 300, 600, 1200, 2400, 4800, 9600, 14.4K, 19.2K, 28.8K, 38.4K, 57.6K, 115.2K . Additional baud rates are also possible.

Data bits: The options are: 5, 6, 7, 8

Parity: The options are: None, Even, Odd, Mark, Space, A/D

Note: A/D parity activates the multi-drop mode and is available only on DataBoy Plus.

Stop bits: The options are: 1, 2

Time stamp: The options are: Off, On

Note: Time-stamps are available only on DataBoy Plus.

Dump rate: This is the baud rate used for dumping captured data via the serial port. The options are the same as for the Baud rate selection (above).


Each time DataBoy is powered on, these parameters are set to default settings that are stored in non-volatile flash memory on the cartridge. The initial factory default settings are 9600 baud, 8 data bits, no parity, 1 stop bit, no time-stamps. These default settings can be modified by pressing the SELECT key in the Setup Page. This gives the user the option to set the power-on defaults to the settings currently selected in the Setup Page.
