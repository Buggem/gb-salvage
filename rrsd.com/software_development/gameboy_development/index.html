Programmer for Gameboy Cartridges using Jeff's C3 progammer

I made this program in order to program Gameboy cartriges with Flash using Jeff's C3 cartridge. It's set up for a particular Parallel Port (0x0378). Changing this would require a recompile. It depends upon a hardware device driver DLL whose source is included in a 3MB download.
